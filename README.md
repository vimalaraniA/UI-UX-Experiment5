## Aim:

To understand and apply the process of identifying user experience (UX) problems, selecting appropriate research methods, and developing user personas for different project scenarios such as a health & fitness app and an e-government service portal.

## Algorithm:

Step 1: Select two different UX project scenarios (for example, a health app and an e-government portal).

Step 2: Identify the major user pain points or usability challenges in each scenario.
